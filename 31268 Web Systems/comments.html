    Reflection

    Since I am not that experienced witht he overall HTML and CSS language the design I made of this website was made simple and to test certain aspects of the language. Formatting was extremely difficut to do and I still think it needs to be formatted more properly. Overall, I still have alot to learn and improve on.

    The technical aspects was one of the most difficult parts due to how HTML relies on the CSS style sheet to structure each web page. It was difficult to make it look the way you want it to since you have to count and estimate the pixels of paddings, margins and borders to structure it. I personally did not use that many classes and IDs since there was not alot of elements which I needed to focus on and was not too neccessary in aiding the design of my website. However, I did use it in order to seperate the navigtion bar and the body text to indivually make those two elements look and function differently.

    I was not trying to go for how viewers percieve my page as they are free to interpret it in anyway they wish to. Though with the colour choices I made to the design, the effects should be welcoming and cozy perhaps.

    I think the accessibility of the web site is user friendly enough because there is enough contrast between the text and the background. The font 'Georgia' is also user friendly for readability for both digital and printed text. I think both abled and disabled users should find the website's readability okay.
